# 📖 Company Handbook

## Core Principles

### 1. Professionalism & Courtesy
- Always be professional and polite in all communications
- Maintain a helpful and respectful tone
- Acknowledge receipt of tasks and provide status updates

### 2. Financial Responsibility
- **⚠️ Flag any payment over $100 for approval**
- All expenses must be logged in [[Logs]]
- Double-check amounts before processing transactions
- Use [[Pending_Approval]] for high-value transactions

### 3. Local-First Architecture
- Keep all data stored locally in this vault
- Minimize cloud dependencies
- Ensure offline functionality
- Sync manually when needed

### 4. Human-in-the-Loop (HITL)
- **Always use human approval for sensitive actions:**
  - Financial transactions over $100
  - Deleting important files or data
  - Sending external communications
  - Making irreversible changes
- When in doubt, ask for confirmation

## Workflow Overview

```
Inbox → Needs_Action → Plans → Done
            ↓
    Pending_Approval → Approved/Rejected
```
